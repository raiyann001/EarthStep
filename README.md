
# 🌍 EarthStep

**Every Step Shapes Our Future.**

EarthStep is a sustainability-focused web platform that encourages people to build environmentally friendly habits through daily eco-challenges, rewards, achievements, and community participation. The project aims to inspire individuals to take small actions every day that collectively create a positive impact on our planet.

---

## ✨ Features

* 🔐 User Authentication
* 🌱 Daily Eco Challenges
* ⭐ XP & Level System
* 🏅 Achievement Badges
* 🔥 Daily Streak Tracking
* 🏆 Community Leaderboard
* 👤 User Profile
* 🌍 Environmental Awareness
* 📚 Research & Educational Resources
* 🤝 Community Engagement

---

## 🛠️ Technologies

* HTML5
* CSS3
* JavaScript
* Firebase Authentication
* Cloud Firestore
* Firebase Hosting

## 🎯 Project Goals

* Promote sustainable daily habits.

* Make sustainability engaging and accessible for everyone.
